# Ember

Ember is a premium open-source native media player built entirely in Kotlin and Jetpack Compose. Enjoy beautiful dark-themed playback, high performance, and advanced audio/video processing controls.

**This project is actively developed. Please report any bugs you find in the [Issues](https://github.com/anilbeesetti/nextplayer/issues) section.**

## Features

- **Modern Premium Interface**: Built with Kotlin and Jetpack Compose with a sleek dark-themed UI.
- **AMOLED Black Canvas**: True black background optimization for OLED screens.
- **Advanced Audio Processing**: AI Enhancer and Dialogue Booster controls for crystal-clear vocals.
- **Intuitive Interactions**:
  - Step-wise pinch to cycle aspect ratio.
  - Vertical swipe to change brightness (left) / volume (right).
  - Horizontal swipe to seek.
  - Item entrance transitions for a fluid UI feel.
  - App animations toggle to customize transition speeds.
- **Media Organization**: Title and file sanitizers to clean up video file names.
- **Material 3**: Complete support for dynamic colors.
- **Tree, Folder, and File View Modes**: Custom layout configurations for media library navigation.
- **Wide Format Support**: Hardware and software decoding options for all modern video/audio formats.
- **Additional Features**: Picture-in-picture mode, zoom gesture, external subtitle and audio track selection, and playback speed controls.

## Supported formats

- **Video**: H.263, H.264 AVC (Baseline Profile; Main Profile on Android 6+), H.265 HEVC, MPEG-4 SP, VP8, VP9, AV1
- **Audio**: Vorbis, Opus, FLAC, ALAC, PCM/WAVE (μ-law, A-law), MP1, MP2, MP3, AMR (NB, WB), AAC (LC, ELD, HE; xHE on Android 9+), AC-3, E-AC-3, DTS, DTS-HD, TrueHD
- **Subtitles**: SRT, SSA, ASS, TTML, VTT, DVB
